Game Rules

Malefiz is a board gameboard for 2 to 4 players. Each player has 5 figures to play with. Every figure starts in their base, at the bottom of the board. The beginning player rolls the dice. The thrown number has to be pulled completly. While walking with the figure, change of direction is not allowed. If a player lands on an enemys figure, the enemy has to put his figure back to his base. Figures are allowed to jump over other figures, but not over barricades. If a player lands on the barricade (with the exact number), he has to set the barricade anywhere on the gameboard board, all black fields are allowed, except the lowest row.

Goal Of The Game

The player reaching the top of the gameboard first (with the exact number rolled with the dice) with one figure wins the gameboard.

Informations

When the player does a faulty move, the gamefigure gets reset to the starting point of its move (ex. walk into a barrier with one or more moves left). This is done with our Undo-Manager.

If a player can't do a regular move, the player can press the 'skip' button to skip his move.

Extra Features

Skip A Move!

The player is allowed to skip his current move. It will be the next players turn.

TUI Command -> 'skip'

Save The Game Progress!

The player is allowed to save the gameboard, when he finished his current move and before the next player throws the dice.

TUI Command -> 'save'

Load The Game Progress!

The player is allowed to load the savegame, before the gameboard starts. You need to add enough players before loading!

TUI Command -> 'load'

Undo The Last Move!

The player is allowed to undo his last movement! (And also for information: The whole move gets undo'd if the player moves into a not allowed direction)

TUI Command -> 'undo'

Redo The Last Move!

The player is allowed to load the savegame, before the gameboard starts. You need to add enough players before loading!

TUI Command -> 'redo'

Reset The Game!

The player is allowed to reset the gameboard at any given time via GUI! This resets the whole gameboard and deletes all players.

TUI Commands Information

After the player has rolled the dice, a number from 1 to 5 has to be entered, to choose the gamefigure.
